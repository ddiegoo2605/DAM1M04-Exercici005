Imatge giratoria

Descobreix qui hi ha a la casa. (La imatge gira quan el cursor està a sobre)

Estat inicial:

.flip-inner{transition: transform 0.6s;transform-style:preserve-3d;}

Acció al pasar el cursor a sobre:

.flip-card:hover .flip-inner{transform: rotateY(180deg);}

Acció que retorna al estat inicial:

.flip-back{transform: rotateY(180deg);}

Menú secundari

Opcions amb transició de color i letter-spacing.

.menu-secundari a{ font-family: "Stack Sans Notch", sans-serif; font-size: 22px; text-decoration: none; color: black; /* ESTADO INICIAL */ letter-spacing: 0px; transition: color 0.4s ease, letter-spacing 0.4s ease; }

/* HOVER — color + separación entre letras */
.menu-secundari a:hover{ color: white; letter-spacing: 3px; }
